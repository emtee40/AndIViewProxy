# AndIViewProxy

An apk that runs the node cos

The telly is on the internet now

yeah it is

rolled from : <https://github.com/OpenGovAus/iview-proxy>

more info here : <https://github.com/AusOpenTech/ABCData/>

What is a tracker? : <https://reports.exodus-privacy.eu.org/en/info/trackers/>


**INSTRUCTIONS**

- run the app
- press the START button to start the proxy server
- press the LAUNCH button to open device web browser
- enter program name in the search box
- watch the telly your taxes already paid for


currently build target API 30, minimum is API 22

Release version is a signed APK, very low-brow app with no checks

Currently has boilerplate permission of INTERNET but its the device browser that uses it

Some on-screen easy-peasey instructions

**TRACKERS**

The following corporate trackers are bypassed by using this proxy
as per Exodus Privacy latest report: <https://reports.exodus-privacy.eu.org/en/reports/182856/>

- Youbora (NPAW)
- Facebook Analytics
- Facebook Login
- Gigya (SAP)
- Google Analytics
- Google CrashLytics
- Google Firebase Analytics
- Google Tag Manager
- Nielsen
- OzTAM
- Snowplow
- Tealium
